# Smara Landing Page

Landing page for **Smara** — small-batch, all-natural perfume handcrafted from organic botanicals.

Built with Astro and Tailwind CSS. Based on the [revisita](https://github.com/erfianugrah/revisita) theme.

## Development

```sh
npm install
npm run dev      # start the dev server
npm run build    # build the static site to dist/
npm run preview  # preview the production build
```
